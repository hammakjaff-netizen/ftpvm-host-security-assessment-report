# FTPVM Host Security Assessment and Vulnerability Analysis Report

This repository contains a professional technical assessment report based on a controlled host security analysis of an FTP server virtual machine.

## Summary
The report covers:
- FTP authentication and brute-force exposure testing
- Anonymous FTP access validation
- Nmap banner, vulnerability, and full TCP scan analysis
- Nikto web server assessment
- Validation of file access, download, and upload behavior
- Analysis of legacy Windows XP and IIS exposure
